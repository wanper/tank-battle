# 🎮 坦克大战 (Tank Battle)

一个使用 HTML5 Canvas 开发的经典坦克大战游戏。

## 游戏截图

![Tank Battle](screenshot.png)

## 游戏特性

- 🎯 经典坦克大战玩法
- 🎨 像素风格图形
- 🤖 敌人AI智能追踪
- 🏆 多关卡挑战
- 💥 砖墙/钢墙/水域等多种地形
- 🛡️ 保护基地不被摧毁

## 如何游玩

### 在线游玩
直接在浏览器中打开 `index.html` 文件即可开始游戏。

### 操作说明

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 发射子弹 |
| P | 暂停游戏 |

## 游戏规则

1. **目标**: 消灭所有敌方坦克，保护你的基地
2. **生命**: 玩家有3条生命
3. **得分**: 每消灭一个敌人获得100分
4. **关卡**: 消灭所有敌人后进入下一关，敌人数量会增加
5. **失败条件**: 生命耗尽或基地被摧毁

## 地形说明

- 🧱 **砖墙**: 可被子弹摧毁
- 🔩 **钢墙**: 无法被摧毁
- 💧 **水域**: 坦克无法通过

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/tank-battle.git

# 进入目录
cd tank-battle

# 用浏览器打开 index.html
```
